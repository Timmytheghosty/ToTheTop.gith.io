To The Top!

This game is to the top a Platformer that gets harder as the game progresses. The goal of the game is to reach the top of the level before the timer ends. Be careful when jumping because if you miss a jump you might have a great fall. Controls are simple for downloaded game arrow keys to move and spacebar to jump. For mobile devices click on the buttons on the bottom of the screen. Game is attended for all audiences

Game uses assets from a free comicanical use assest pack Assest pack is called Sunny Land - Pixel Game Art Collection by ansimuz and can be found on itch.io
